Job Portal (MERN Stack)

A full-stack Job Portal Web Application built using the MERN stack, enabling seamless interaction between job seekers (students) and recruiters with role-based dashboards and secure authentication.

📌 Features
👨‍🎓 Student (Job Seeker)
JWT-based authentication (Register/Login)
Profile management (bio, skills, profile picture, resume upload)
Browse and apply for jobs
Track application status via dashboard

🧑‍💼 Recruiter
Secure authentication with role-based access
Create and manage companies
Post and manage job listings
View applications and accept/reject candidates

🛠️ Tech Stack
Frontend: React.js, Redux, Tailwind CSS, ShadCN UI
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT (JSON Web Tokens)
File Storage: Cloudinary
API Testing: Postman

⚙️ Key Highlights
Implemented role-based authentication & authorization
Built RESTful APIs for job postings and applications
Integrated Cloudinary for resume and image uploads
Designed responsive UI using Tailwind CSS
Managed application state using Redux
